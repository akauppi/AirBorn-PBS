# Kotka PBS

This page tells how to build and deploy the documentation.

## Requirements

- npm

```
$ npx npm-force-resolutions
$ npm install
```

## Getting started

```
$ npm run dev
...
```

Open [http://localhost:3100](http://localhost:3100) to see the building guide.
